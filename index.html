    Difficulty Editor Tool

    This Tool was built in the Unity Engine using C#.

    Progressive Difficulty Editor Tool is a project settings tool under windows in the Unity Engine that monitors the values of the playthrough of the game. Then changes the Enemy AI within the game by increasing difficulty of the enemy and then persistently saves those changes for the next play through when the player dies.

    FEATURES

    • GUI Interface
    • Manual Adapatable Difficulty Settings
    • Persistantly changes Enemy Difficulty
    • Working Reset button
    • More and Improved Multipliers
    • Better GUI
    • Release Tool/Monetisation Prospects

    DEMO

    During my Final Year project, The main goal was to create this Progressive Difficulty Editor Tool, which meant some of the game that was designed to show case it lacked in a better framework. The tool was difficult for me to develop at first I struggled to find research online in this area to help develop it, probably caused more by the fact i was looking for the wrong thing and using the wrong terminolgy during the early process. I kept getting confused about the difference of just a standard in game difficulty system, opposed to creating an external tool to monitor the difficulty.


    Max enemies in the scene just monitors the Constant value of the predetermined amount per wave

    Dmg Multiplier just actively monitors the wave and then applies the multiplier for the next playthrough increasing the difficulty for the player

    CTF-The-Oasis

    This Level Design was built in the Unreal 4 Engine using all assets in the Editor.

    CTF The OASIS is a Capture the Flag map I made in Unreal Tournament 4 editor in my first year of university. All the assets came with the editor we were told to use, we had to go through three iterations of the design process, first a simple drawing and then build the drawing then in the second interation we would apply the weapon and health drops to help understand balance and game flow, then a final interaion of the map with all of these finessed ready for showcase .

    Level Design

    Below is the first iteration of the Level Design, due to the nature of Capture the flag everything has to be very well balanced and symmetrical. I decided to take that process a bit further in terms of the map design and decided upon lush oasis green vs the bleak and barron desert gold.


    • Symmetrical
    • Good Weapon placement Areas
    • Good Health/Shield Placement Areas
    • Can't really Snipe
    • Too cramped, no high risk, high reward areas
    • Didnt flow smoothly
    • Lack of height

    Below is the 2nd iteration of the Level Design process. As you can see already its a lot more open to allow for more freedom of play and various heights to add to the player experiece

    • Symmetrical
    • Much more open.
    • New interesting gameplay areas for the player
    • Various heights to allow for better line of sight
    • High risk high reward areas
    • More Buildings
    • Utilising even further height
    • Secret area
      • More shield and rare weapon placements
    • Scale improvements
